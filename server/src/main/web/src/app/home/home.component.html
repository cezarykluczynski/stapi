Welcome to STAPI, a Star Trek API!

What is it

  • STAPI is the first public Star Trek API, accessible via REST.
  • STAPI serves a read-only model of all things Star Trek, including characters, performers, species, episodes, spacecrafts, books, astronomical objects, video releases, and more.
  • STAPI is an open source project, that anyone can contribute to.
  • STAPI is now in beta version. The API model is considered stable at this point, and when it is evolving, it is evolving in a non-breaking manner, using API versioning.

How it works

  • STAPI uses publicly available sources, mainly Memory Alpha, to get its data, but it also uses Star Trek Cards and Memory Beta.
  • Data is cleansed, standardized and put into relational model.
  • API is publicly available via REST, and can be browsed using an API Browser.
  • Swagger specs can be downloaded.
  • STAPI is written in Java, with tests written in Groovy, and frontend written in TypeScript.
  • Clients are available in several languages.
