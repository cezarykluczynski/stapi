API documentation is a single Swagger spec, accessible here. You can open it in Swagger Editor directly.

If that doesn't work, because you're running STAPI locally or HTTPS is not available, then just copy contract content to editor window.

In the editor, clients can be generated for variety of programming languages.

In case of any problems with Swagger specs, please report an issue.

If you plan on maintaing STAPI client in language other than Java, please open an issue and let others now. Clients are listed on STAPI website.
